Zadatak
=========
Pokrenite "http-server" u tekućem direktorijumu.

Otvoriti localhost:8080/ stranicu u web čitaču.

Uneti brojeve u polje i uočiti ispis koji je pod filterom.

Ugasiti server i uneti komandu "npm install"

Pokrenuti karma test sa komandom "karma start tests/karma.config.js"
Napomena: Ukoliko terminal kaže da ne prepoznaje komandu karma potrebno je izvršiti komandu "npm install -g karma-cli"

Sačekati da test prođe i otvoriti datoteku secondsToTime.filter.spec.js i protumačiti kod.
