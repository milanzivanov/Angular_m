Zadatak
=========
Pokrenite "http-server" u tekućem direktorijumu.

Otvoriti localhost:8080/ stranicu u web čitaču.

Ispitati ponašanje aplikacije.

Otvoriti employee.spec.js u tests/e2e fascikli i proučiti testove.

Napomena: Potrebno je da selenium server bude pokrenut (webdriver-manager), kao i HTTP server (http-server).

Napomena 2: Putem testa se upisuje u mongolab slog, i kako je podešen validator za jedinstven JMBG test neće proći osim ako sami ne izmenite verdnost za jmbg koja se unosi i očekuje u neku jedinstvenu vrednost.

Napomena 3: Upotrebom 'element' i 'all' funkcije nad ElementFinder objektom (element koji smo prethodno locirali) vrši pretragu nad decom prethodno nađenog elementa.
