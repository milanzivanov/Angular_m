Zadatak
=========
Izvršiti sledeće komande u tekućem direktorijumu:

"npm install karma --save-dev" 	- instalira Karmu za tekući projekat i čuve u package.json zavisnost
"npm install -g karma-cli"	- potrebno je izvršiti samo jednom da se na računaru instalira Karma
"karma init"			- pokreće wizard za izradu karma.config.js datoteke

Ispratiti rad wizarda i obratiti pažnju na sve korake. Nakon generisanja karma.config.js datoteke ispitati njen sadržaj.

Izvršiti komandu "npm install karma-jasmine@2_0 --save-dev". Koristimo Jasmine 2.0 zbog naprednih funkcionalnosti.
