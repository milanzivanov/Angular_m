Zadatak
=========
Pokrenite "http-server" u tekućem direktorijumu.

Otvoriti localhost:8080/ stranicu u web čitaču.

Ispitati ponašanje aplikacije.

Pokrenuti testove sa protractor tests/protractor.config.js

Otvoriti employee.spec.js u tests/e2e fascikli i proučiti testove.

Napomena: Potrebno je da selenium server bude pokrenut (webdriver-manager), kao i HTTP server (http-server).
