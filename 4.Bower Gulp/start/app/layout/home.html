Assignment

Make sure you read the About page before starting your assignment.
Your assignment will be to take what we've learned today and apply it to your project.
Note: take special care to include all the javascript files in your index.html

Complete the following exercises:
  1. Go through all the javascript files and make sure they adhere to the best practices of writing angular components:
    1. Separate the code in different files so as to adhere to the "One component per file" rule (Note: be mindful of the naming convention used for file names);
    2. Use self-invoking functions wherever necessary;
    3. Use controller as sintax instead of the $scope object;
    4. Inject your dependencies using the inline array or via the $inject property annotation;
    5. Bonus: create a nested view structure described in the previous lesson using the ui-router;
    6. Bonus: modify the add/edit form of both companies and employees so that the company/employee is retrieved/created during routing instead of inside the controller (Note: use the resolve property of the state object);
    7. Bonus: handle the case when an employee/company is requested but the database doesn't have the employee/company with the given id, e.g. display an alert and/or redirect to the home page;
  2. Create a directory structure that adheres to the LIFT principle as well as the "Segment by application component" practice (Note: if you haven't done this in the previous step, make sure to adhere to the "One module per component" principle);
  3. Replace the $http service with the $resource service;
  4. Replace all external javascript dependencies and libraries with bower dependencies:
    1. Form a bower.json file which has all the basic fields field out (Note: use the "bower init" command);
    2. Fill in all the external dependencies;
    3. Create a .bowerrc file to change the name of the directory where to download the dependencies, e.g. assets/js;
  5. Use the gulpfile.js file from the last example of this lesson and modify it so that it works for your application (Note: make sure to modify the index.html file);
  6. Bonus: create SASS files to edit the style of the application and expand the gulpfile.js file with plugins that compile SASS and minify CSS and incorporate them in your build process;
  7. Bonus: read about JSHint and expand your gulpfile.js file with the appropriate plugin which you will incorporate in your build process;
  8. Expand the application by adding a new component which acts as a place registry (Note: Since it's hard to debug minified files you should avoid using gulp (or at least the minification plugin) until you've developed and tested your code):
    1. A place has two attributes, a name and a postal code, e.g. {name:'Novi Sad', code:'21000'} (Note: make sure to set the _id field of the entity to either name or code before persisting it);
    2. A form to view all the registered places, which includes sort, search and pagination functionality;
    3. A form to add/edit a place;
    4. Update the side bar to include a link to the place registry;
    5. Expand the company add/edit form so that there is a field named "headquarters" where the component for that field is a combobox which contains all the places (Note: combine ng-repeat with the option tag nested in the select tag to create a combobox);
    6. Bonus: repeat the above process for the employee by adding a "placeOfBirth" field;
    7. Bonus: expand the employee list view so that it displays the "placeOfBirth" field and searching by place (Note: use a combobox to search, so that you can either have employees from one place displayed or from all of them);
  9. Bonus: add a "Revert" button on the add/edit form of the company, employee and place entity. What this button does is negate the changes you've made to the current company/employee/place, restoring the selected entity to it's initial state while remaining on the page (Note: read about angular.copy and use it to form a clone object when the form opens);
  • Focus on doing non-bonus excercises during class;
  • What you don't complete during the class remains as a homework assignment;
  • It is highly recommended that you complete each of the listed excercises, including bonus excercises;
