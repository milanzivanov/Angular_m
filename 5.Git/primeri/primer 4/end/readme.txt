Zadatak
========
Ukucati http-server komandu u terminal u tekućem direktorijumu.

Otvoriti aplikaciju u web čitaču na localhost:8080/ i proći kroz korisnički interfejs.

Unaprediti formu za unos radnika koristeći angular UI Bootstrap biblioteku:
  - Datum rođenja treba da se bira putem datepicker-a
  - Pored mesta rođenja treba da stoji dugme koje otvara modalni dialog za unos novog mesta
